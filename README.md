# DOIT

## A simple to-do list developped to practice some React Native's concepts reviewed by Rocket Seat's professional boot camp.

### Technologies:

- Expo
- React Native
- Typescript

### Run it:

- Donwload the code
- Inside the folder DOIT run:
  npm i
- then run:
  expo start
- scan the QR code and have fun!
